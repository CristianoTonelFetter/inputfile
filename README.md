# inputfile
<div>
    JQuery plugin for custom input file. 
</div>
----------

<div>
Install:

Bower 

$ bower install jquery-inputfile

</div>
----------

<div>
Usage: 

HTML:

&lt;div id="myelement"&gt;&lt;/div&gt;

Javascript:

$("#myelement").inputfile();
----------

or
----------

$("#myelement").inputfile({
    customOffset: { top: 0, left: 0 }, 
    onChange: function(){}
});

</div>
